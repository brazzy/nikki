nikki by brazzy

GPS Photo diary - standalone Java app, geotagging, export to KMZ

Nikki (from Japanese 日記, "diary") enables you to share travel experiences with your friends and family. It is a standalone (Java) application that * Reads photographs and GPS track files * Assigns each image a GPS location (based on its timestamp) * Allows you to annotate each photo with a title and description text * Exports annotated images and GPS tracks for each day to a single, compact KMZ file that can be viewed in Google Earth

Dependencies

Requires a Java runtime to run, everything else is included. To develop and build, you need an IDE that also supports Groovy. I suggest eclipse with the SpringSource Tool Suite (STS) plugin.

Install

No installation required, just download the JAR from the download page. On most systems with a Java runtime installed, it should run when double-clicked. Otherwise, use the command line, go to the download directory and type "java -jar Nikki.jar"
