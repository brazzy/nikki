Teile deine Reiseerlebnisse mit der ganzen Welt! Die wichtigsten Arbeitsschritte:

  1. Füge ein Verzeichnis mit Bildern und GPS-Logs hinzu (bisher nur NMEA-Format)
  2. Durchsuche Verzeichnis, gebe die Zeitzone ein, nach der die Uhr der Kamera gestellt war.
  3. Suche Bilder zum Exportieren aus, füge Titel und Beschreibung hinzu
  4. Suche ein Bild mit genau bekanntem Ort und finde die Zeitabweichung
  5. Geotagge die Bilder (weist anhand der Zeitstempel GPS-Koordinaten zu)
  6. Exportiere Bilder und GPS-Pfade in eine kompakte KMZ-Datei zur Anzeige in Google Earth
  7. Speichere alle veränderten Daten in den EXIF-Headern der Bilddateien

Erstellt mit Groovy, Java und folgenden Bibliotheken: RouteConverter, MediaUtil, Joda Time, Java API for KML,
Java Spatial Index, Java Image Scaling, dem Silk icon set, und der Genonames.org-Datenbank.
