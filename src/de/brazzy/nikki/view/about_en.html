Share your travel experiences with the world! Basic workflow:

  1. Add directory containing images and GPS logs (NMEA format only so far)
  2. Scan directory, enter time zone to which camera time was set
  3. Select images to export, add titles and descriptions
  4. Select image with exactly known location and find camera time offset
  5. Geotag images (assigns GPS coordinates based on time)
  6. Export images and GPS paths to compact KMZ file for display in Google Earth
  7. Save all changed data to the image files' EXIF headers

Built using Java, Groovy, and the following libraries: RouteConverter, MediaUtil, Joda Time, Java API for KML,
Java Spatial Index, Java Image Scaling, the Silk icon set, and the Genonames.org database.
